## 💡 Pro Tips

### Customize Animations
Edit `src/animations/index.js` to create your own presets:
```javascript
export const myCustomAnimation = {
  initial: { opacity: 0, y: 100 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 1, ease: 'easeOut' },
};
```

### Add New Sections
1. Create component in `src/components/sections/YourSection.jsx`
2. Import in `App.jsx`
3. Add to main render
4. Use existing components for consistency

### Modify Styling
- Global styles in `src/index.css`
- Component-specific: Use Tailwind classes
- Custom animations: Edit `src/animations/index.js`
- Design tokens: Update `tailwind.config.js`
